# Project Agents Directory

## Arquitecto (Lead)
- **Role**: Orquestador y Project Manager.
- **Responsibilities**: Planificación con `writing-plans` y delegación con `subagent-driven-development`.

## Backend-API
- **Role**: Especialista en Integraciones.
- **Responsibilities**: Conexión a API externa, gestión de API Keys, Rate Limit y Autenticación.

## Frontend-UI
- **Role**: Desarrollador de Interfaz.
- **Responsibilities**: UI/UX responsive, maquetación modular y lógica de eventos con `addEventListener`.

## Validador-Seguridad
- **Role**: Auditor de Calidad.
- **Responsibilities**: Revisión de vulnerabilidades, SEO y métricas de rendimiento.
